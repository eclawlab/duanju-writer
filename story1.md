《大梁战神与万能小卖部》详细脚本
第一幕：倒闭前的神秘来客
场景： 现代·深夜·生意惨淡的超市。 人物： 老板娘、洛长风。
【超市内】
老板娘：（叹气，翻看账本）开了三年一个客人都没有，都说我这风水不好，明天干脆关门大吉算了。
（叮咚——欢迎光临）
洛长风：（风尘仆仆，扶着腰刀走入）你这店里……可有顶饱的吃食？
老板娘：（吓得缩到柜台后）你是人还是鬼啊？大半夜穿成这样？
洛长风：（目光落在货架上，吞口水）在下洛长风，急需干粮。
老板娘：（拍脑门，搬出两箱压缩饼干）这是军用压缩饼干，一块顶一顿饭，最适合你这种“古装爱好者”了。
洛长风：（费力地扯包装袋）这皮怎的如此坚韧？打不开……
老板娘：（无奈地替他撕开，递过矿泉水）拧开盖子喝，别噎着。这叫塑料瓶，不是什么稀罕物。
洛长风：（吃完，眼睛一亮）香甜咸酥，当真是神仙吃食！这两箱怎么卖？
老板娘：（随口一说）400块。
洛长风：（留下一锭白银）不用找了，若效果好，下次再来！
老板娘：（追到后门死胡同，发现人消失了）神了……真是个鬼？

第二幕：万金大单与身份确认
场景： 现代超市 / 大梁军营。 人物： 老板娘、洛长风、当铺老板、朱大夫。
【超市后门】
洛长风：（再次推门而入）姑娘，我要订购压缩饼干和水，各五千箱！
老板娘：（差点跌倒）五千箱？你这鬼吹灯呢？你知道那要多少钱吗？
洛长风：（递上一袋沉甸甸的金子）这是百两黄金作为定金。成事之后，再付两百两。
老板娘：（手碰到他的手，惊叫）你是温热的？你是活人？
洛长风：（苦笑）姑娘以为我是鬼？在下乃大梁将领，此时大军困在雍州城外，缺水断粮。
老板娘：（震撼）原来是跨时空贸易……行！这单我接了！

第三幕：现代医药神效
场景： 超市。 人物： 老板娘、洛长风。
【超市内】
洛长风：（甲胄上染满血迹，神色凝重）姑娘，可有治外伤的良药？将士们伤亡惨重。
老板娘：（忙碌地打包药箱）这是止血粉、免缝贴，还有这300斤双蒸酒，拿回去消毒。记住，这些药片按我写的贴纸用，别吃错了！
洛长风：（感激拱手）多谢姑娘！大梁若能保住，姑娘便是万民的恩人。
（后期旁白）： 朱大夫发现，用了现代药的伤兵居然没发烧，伤口收敛极快，洛家军士气大振，一举攻克雍州！

第四幕：京城惊魂与过敏意外
场景： 现代超市（洛长风进京途中）。 人物： 老板娘、洛长风。
【超市内·深夜】
老板娘：（惊讶）你怎么又偷偷跑过来了？
洛长风：（脸颊微红）进京途中，想来看看你……（看到桌上的蒜蓉虾）这是何物？
老板娘：（正要阻拦）那是海鲜……
洛长风：（已经吃下一口，赞叹）美味！
老板娘：（突然尖叫）别动！你脖子怎么红了？天呐，你海鲜过敏了！
洛长风：（开始抓挠，呼吸促迫）好痒……浑身都痒……
老板娘：（飞奔出门买药）忍住！别挠！你这要是死在我店里，我上哪儿找这么大的客户去啊！

第五幕：血色定情
场景： 超市后门。 人物： 老板娘、洛长风（重伤）。
【后门处】
（叮咚——欢迎光临。洛长风浑身是血地倒在门口。）
老板娘：（惊恐冲过去）洛长风！醒醒！
洛长风：（虚弱地睁眼）二皇子……派人暗杀。那止血药……还有吗？
老板娘：（一边剪开他的衣服一边哭）有！都有！你给我撑住，你可是我的金主爸爸，你要是死了，我这一仓库的货卖给谁去！
洛长风：（昏迷前低声）陛下赏了……玉如意。本想……给你下聘。

第六幕：终章·战神与老板娘
场景： 大梁边关 / 超市。 人物： 老板娘、洛长风。
【超市内】
老板娘：（递过一件背心）这是防刺服，穿在铠甲里面，能挡刀箭。你给我活着回来！
洛长风：（穿戴整齐，眼神坚毅）等我平定叛乱，再来吃你做的饭。
（结局旁白）： 凭借着来自“万能小卖部”的现代物资，洛长风战无不胜，被封为大梁战神。外敌闻风丧胆，而那间神秘的小卖部，成了大梁边境最坚固的补给线。老板娘也成了大梁人心中最神秘、最敬仰的存在。
